# Freemium Funnel

## Kurzbeschreibung
Der Freemium Funnel ist ein strategisches Framework zur Entwicklung von kostenlosen Produktversionen, die Nutzer schrittweise zu zahlenden Kunden konvertieren. Die Methode hilft dabei, die richtige Balance zwischen kostenlosem Wert und Premium-Features zu finden.

## Einsatzzweck
Diese Methode eignet sich für digitale Produkte und Services, die über ein Freemium-Geschäftsmodell monetarisiert werden sollen. Besonders wertvoll in der Produktstrategie-Phase, wenn du die Grenzen zwischen kostenloser und kostenpflichtiger Version definieren musst.

## Kurzanleitung
1. **Kernwert identifizieren:** Definiere den Hauptnutzen deines Produkts, den auch Free-User erhalten sollen
2. **User Journey mappen:** Zeichne die typische Nutzungsreise von der Anmeldung bis zur ersten Kaufentscheidung auf
3. **Friction Points finden:** Identifiziere natürliche Hindernisse, wo User auf Limits stossen → Speicherlimit, Anzahl Projekte, erweiterte Features
4. **Value Ladder erstellen:** Entwickle eine Stufenleiter von kostenlosen zu Premium-Features mit steigendem Wert
5. **Conversion Trigger definieren:** Bestimme konkrete Auslöser für den Upgrade → "Du hast 80% deines kostenlosen Speichers verwendet"
6. **Onboarding optimieren:** Gestalte die ersten Schritte so, dass User schnell Wert erleben und Premium-Features entdecken
7. **Metriken festlegen:** Definiere KPIs wie Conversion Rate, Time-to-Upgrade und Customer Lifetime Value
8. **Testing Framework:** Plane A/B-Tests für verschiedene Freemium-Grenzen und Upgrade-Prompts

## Beispielprompt
{% code overflow="wrap" %}
```
Entwickle einen Freemium Funnel für eine Projektmanagement-App. Definiere: 1) Welche Kernfeatures kostenlos bleiben, 2) Wo natürliche Upgrade-Punkte entstehen, 3) Wie die ersten 7 Tage Onboarding aussehen, um sowohl Wert zu liefern als auch Premium-Features zu demonstrieren.
```
