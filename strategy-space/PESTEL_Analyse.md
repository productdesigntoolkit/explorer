# PESTEL Analyse

## Kurzbeschreibung

Die PESTEL Analyse untersucht systematisch sechs externe Einflussfaktoren (Political, Economic, Social, Technological, Environmental, Legal) auf ein Unternehmen oder Projekt. Sie hilft dabei, Chancen und Risiken im Marktumfeld zu identifizieren und strategische Entscheidungen zu treffen.

## Einsatzzweck

Verwende die PESTEL Analyse zu Beginn von Projekten, bei Marktanalysen oder strategischen Planungen. Sie eignet sich besonders für die Bewertung neuer Märkte, Produkteinführungen oder bei grundlegenden Geschäftsentscheidungen.



## Kurzanleitung

1. **Rahmen definieren:** Bestimme den Analysegegenstand → Produkt, Markt, Unternehmen oder Projekt
2. **Team zusammenstellen:** Lade Experten aus verschiedenen Bereichen ein → Marketing, Legal, Technologie, Finanzen
3. **Political analysieren:** Sammle politische Faktoren → Regulierungen, Handelspolitik, Stabilität, Subventionen
4. **Economic bewerten:** Identifiziere wirtschaftliche Einflüsse → Inflation, Zinsen, Konjunktur, Kaufkraft
5. **Social untersuchen:** Erfasse gesellschaftliche Trends → Demografie, Lebensstil, Bildung, Werte
6. **Technological erfassen:** Analysiere technologische Entwicklungen → Innovation, Automatisierung, F\&E, digitaler Wandel
7. **Environmental prüfen:** Bewerte Umweltfaktoren → Klimawandel, Nachhaltigkeit, Ressourcenknappheit, Umweltauflagen
8. **Legal dokumentieren:** Sammle rechtliche Aspekte → Gesetze, Standards, Compliance, Patente
9. **Auswirkungen bewerten:** Bewerte jeden Faktor nach Einfluss und Wahrscheinlichkeit auf einer Skala von 1-5
10. **Massnahmen ableiten:** Entwickle konkrete Handlungsempfehlungen für die wichtigsten identifizierten Faktoren

## Beispielprompt

{% code overflow="wrap" %}
```
Führe eine PESTEL Analyse für ein E-Bike-Startup durch, das in der Schweiz und Deutschland expandieren möchte. Analysiere alle sechs Faktoren und bewerte deren Einfluss auf das Geschäftsmodell. Identifiziere die drei wichtigsten Chancen und Risiken.
```
